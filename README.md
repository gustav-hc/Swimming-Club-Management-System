
# Swimming Club Management System

A Java-based administrative system developed as a first semester project in the Datamatiker programme at Erhvervsakademi København.

## Description

The system was developed for a swimming club and supports administration of members, finances and training activities.

The application is console-based and was developed in a team of four students using Scrum principles and Git for version control.

## Features

### Chairman (Formand)

* Create new members
* View all members
* Update membership fees

### Treasurer (Kasserer)

* Register member payments
* View members in arrears
* View expected membership income

### Coach (Træner)

* View top 5 swimmers
* Register training results
* Register competition results
* Assign competition swimmer status

### Data Persistence

* Save data to files
* Load saved data when the application starts
